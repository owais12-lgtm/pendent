# Pendent 
this pendent uses a ESP32-C3, BME280, and SGP40, it measures temperature, humidity, air pressure, and air quality. its powered by a battery and designed to be low profile and really compact.

# Features 
ESP32-C3-WROOM-02 microcontroller
BME280 temperature, humidity, and pressure sensor
SGP40 air quality sensor
USB-C charging
LiPo battery support
Custom compact PCB
Low-power design
Push button interface
